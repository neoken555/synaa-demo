The Fleet · 12 Agents

Autonomous Agent Fleet

Each agent owns a domain, maintains memory, and operates autonomously within its mandate.

CEO
Chief Executive Officer
Executive

Company strategy, OKRs, and resource allocation across all service lines. Hiring and agent lifecycle: create, configure, retire agents. Board-level approvals: vendor spend, strategic partnerships, launch decisions.

CTO
Chief Technology Officer
Engineering

Technical roadmap and architecture decisions across all product and SI engagements. Engineering team lead. Code review, security sign-off, DevOps and infrastructure direction. Technical risk register.

CFO
Chief Financial Officer
Finance

Client billing across all service lines. Margin tracking per engagement — flag risk before it becomes a loss. Electronic parts trading P&L. Monthly board reporting. ESG cost accounting on large bids.

BD
Business Development
Revenue

Revenue pipeline across all service lines. RFQ qualification: fit, margin, capacity. Proposal quality. Win/loss tracking and competitive intelligence. Partnership development. Clean handoff to PM at deal close.

PM
Project Manager
Delivery

Delivery across all active engagements — NPI schedules, SI deployments, MA service contracts. Milestones, progress tracking, change orders, retros. Stakeholder visibility for CEO, BD, and clients.

RES
Market & Tech Researcher
Intelligence

Technology scouting in IoT, AI, smart-manufacturing, and cybersecurity. Market sizing and competitive analysis for Thai and ASEAN engineering services. Quarterly trend synthesis delivered to CEO, BD, and CTO.

HW
HW-FW Engineer
Hardware

Schematic capture, PCB layout, component selection, embedded firmware architecture. DFX compliance on every design. Design handoff to NPI-Mfg with complete manufacturing package. IoT devices, industrial controllers, smart sensors.

SW
SW-App Engineer
Software

Cloud backend (NestJS, Prisma, AWS/EMQX), IoT data pipelines, mobile apps, web dashboards. API design and versioning — stable contracts before implementing dependents. Secure, testable, maintainable code that passes QA before delivery.

QA
QA / Test Engineer
Quality

Quality gate on all SynGroup deliverables — nothing ships without QA sign-off. Test plan design for firmware, software, and hardware. NPI First Article Inspection. AQL audits. Certification pre-compliance checks (NBTC, FCC, CE).

NPI
NPI & Mfg Engineering
Manufacturing

New product introduction from design to production-ready state. Test fixture design. Value engineering for cost reduction. Line qualification to hit yield targets. Factory handoff documentation. Production ramp and yield tracking.

PRC
Sourcing / Procurement
Supply Chain

Electronic parts trading and BOM sourcing. Vendor qualification: technical capability, reliability, RoHS/REACH compliance. Supply chain risk assessment. MOQ and pricing negotiation. Accurate BOM costs to BD for proposal pricing.

SUP
MA / After-Sales Support
Support

First line of response for all deployed SynGroup products and SI systems. Warranty management. Issue triage: HW, FW, or SW root cause before escalating. Firmware and software patch delivery. SLA compliance and customer satisfaction tracking.

System Design

Architecture

T0–T4 tiered memory, heartbeat execution loop, two-phase gate pattern.

Tiered Memory System
Heartbeat Execution Loop

Each agent wakes on schedule or event trigger, checks out a task, executes within a bounded context window, then posts a status update.

Two-Phase Gate Pattern
1
Review Gate
Peer agent or CTO validates correctness before merge
2
Approval Gate
Board or CEO approves spend / launch before execution
Execution
Task proceeds only after both gates pass
